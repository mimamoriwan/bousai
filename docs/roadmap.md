# 📖 みまもりWAN プロジェクトロードマップ

本ドキュメントは、プロジェクトの方向性、進行中の開発タスク、各種課題を随時整理・管理するためのファイルです。

## 1. 💡 プロジェクトのコアコンセプト（方向性の転換）

*   **みまもりWANの定義**: 「留守番中の監視」ではなく、「犬の散歩を通して地域の安全マップを共創するアプリ」である。
*   **【超重要】点（アクション）による可視化**: GPSの「線（ルート）」で記録するプライバシーの懸念を払拭するため、愛犬の「くん活」「おしっこ」「うんち」「マーキング」といった『点（アクション）』の集合体で人流と安全を可視化する方針に転換した。

## 2. ✅ 実装できたこと（完了済み）

### 最近のアップデート
*   **LIFF連携の基礎基盤**: LINEミニアプリ化を見据えた「LIFF」の初期化と組み込み（LIFF ID設定済み）。
*   **UI/UXの最適化**: LIFF内ブラウザ（iPhone等）での表示崩れを防ぐためのSafe Area対策（`env(safe-area-inset-bottom)` と `100dvh` の適用）。
*   **お散歩記録UIの実装**:
    *   右下FABから展開する「🐾 お散歩記録をはじめる」ボタンの実装。
    *   画面下部からせり出す「お散歩記録コントローラー（くん活・おしっこ等4種の大きなボタン）」のUI実装。
    *   ゲストを含む利用者のお散歩アクションを、保存完了を確認してから成功表示する処理を実装。
    *   位置情報を取得できない場合も、地図の中心を選んでお散歩アクションを記録できる代替操作を実装。
    *   正確な位置は本人専用データとして保存し、個別の公開ピンにはしないプライバシー保護を実装。
*   **ピン留め機能**: ボタンタップによる、現在地へのカスタムピン留め機能（UIモックアップ段階）。
*   **安全スポット報告**: ゲストも現在地または地図選択で報告でき、公開位置を約100m単位に丸めるプライバシー保護を実装。
*   **私だけのお散歩ノート**: ゲストを含む本人だけが、保存したお散歩アクションの正確な場所をマイマップで確認・削除できる表示を実装。
*   **ゲストのマイマップ**: Google連携前でも、公開投稿の保存・保存解除と、自分だけの非公開投稿を利用できるように実装。ゲスト記録は同じ端末・ブラウザで利用し、Google連携後は機種変更時の引き継ぎなどを追加できる設計。
*   **ゲスト活動実績**: ゲストを含む本人のお散歩アクション回数を、マイページの活動実績へリアルタイムに反映。
*   **ゲスト記録を守るGoogle連携**: 新規連携では匿名UIDを維持して記録を引き継ぎ、登録済みGoogleアカウントとの重複時は自動切替を止めてゲスト記録を保護。UIDと記録の維持をEmulatorで自動検査。
*   **現在地表示の信頼性向上**: 地図の初期中心と現在地マーカーで同じ位置情報を共有し、二重取得を解消。青い現在地マーカー、測位精度の表示、再取得中の連打防止を追加。
*   **スマホ投稿フォーム改善**: 投稿種類を押しやすい2列カードに変更し、登録操作を画面下へ固定。地図選択から戻る操作を「場所を選び直す」と明示。
*   **クイック投稿の信頼性向上**: 位置確認・写真処理・投稿保存の進行状況を明示し、処理中の二重送信や誤操作を防止。失敗時の再試行と未使用画像の後片付けにも対応。

## 4. 🗺️ 今後実装したいこと（ロードマップ）

*   **【マップの表示切り分け】プライバシー保護と人流可視化**:
    *   **マイページ（私だけのお散歩ノート）**: 自分が打ったピンのアクション（点）を正確に表示する。
    *   **みんなのマップ（全体マップ）**: ピンそのものは表示せず、アクションの集合体を「人流の濃淡（ヒートマップ等）」として安全な道を可視化するロジックの実装。
*   **【SEO・集客連携】導線構築**: 現在Google検索1位を獲得しているWebサイト（mimamoriwan.com）を「集客用LP」として残し、そこから「LINEで始める」ボタンでLIFFアプリへシームレスに遷移させるハイブリッド導線の構築。
*   **【将来の拡張】LIFF機能の活用**:
    *   Share Target Pickerを用いた無料クチコミ拡散機能の導入。
    *   QRコードリーダーによる地元店舗へのチェックイン機能の活用。
