# 🐶 みまもりWAN

みんなの歩幅で作る、地域の安全とペットに優しいまちづくり。
お散歩中の危険箇所や新しい発見から、いざという時の防災・避難所情報までを、スマートに共有できる地域密着型のPWAアプリです。

## 💡 アプリ名とコンセプト

**「みまもりWAN」**
「愛犬がお散歩中に何でも口にしてしまう危険」を防ぎたい、という想いから生まれました。
地域の飼い主さんたちを「みまもり隊員」とし、『知っていれば予防できる』リアルな安全情報（ガラスの破片が落ちている、除草剤が撒かれていた等）や防災情報を、日々のお散歩を通じて共有するサービスです。

## 🛠 使用技術（Tech Stack）

### フロントエンド
- **React**: UIライブラリ（コンポーネントベースの開発）
- **Vite**: 高速なビルドツールとローカルサーバー
- **React Router**: SPA（シングルページアプリケーション）のシームレスな画面遷移
- **Leaflet / React-Leaflet**: オープンソースのインタラクティブマップの実装
- **Vanilla CSS**: 軽量かつ柔軟なスタイリングシステム

### バックエンド（BaaS）
- **Firebase Authentication**: Googleアカウントを利用したユーザー認証
- **Cloud Firestore**: リアルタイムなデータベース同期とデータの永続化
- **Cloud Storage**: HEIC変換・圧縮後のスポット画像のアップロード・保管
- **Firebase Hosting**: プロダクトの高速な配信・公開とデプロイ環境

## ✨ 実装済みの主要機能と仕様

- **🔐 認証・権限制御機能**: Googleログインによるセキュアな認証。ゲスト状態での「閲覧のみ」の権限と、ログインユーザー限定の「ピンの投稿・削除・解決等のアクション」を明確にセパレートしています。
- **🗺️ マップ・リアルタイム投稿機能**: タップした地点に新しいスポットのピンを立てられます。カメラロールからの写真添付（HEIC対応/自動圧縮）、カテゴリー（危険・お散歩情報・防災等）の設定が可能で、追加されたピンは他の端末にもリロードなしで即座に同期されます。
- **🔍 フィルター絞り込み**: マップ画面上のタブをタップすることで、特定のカテゴリーのピンのみを瞬時に表示・非表示に切替えられます。
- **⚡ マップ連動・最新情報フィード**: 投稿されたスポットをマップ下部にタイムライン表示。リストをタップすると、マップが対象のピンへ自動でアニメーション移動（見切れないようオフセット処理）し、吹き出しが開きます。
- **💖 コミュニティリアクション**: 他のユーザーの投稿に対して「💖ありがとう」の送信や、既に安全になった危険箇所を「👍解決済（グレー表示化）」にしてマップを最新状態に保つことができます。
- **📖 ガイド画面**: アプリの目的や使い方（「はじめの手引き」）を分かりやすいUIで説明。既存の「ペット用備蓄リスト」や「避難時の心得」などの重いテキストは、アコーディオン形式（タップで展開）ですっきりと収納しています。
- **🐾 マイページ**: ユーザー情報の管理と、ログイン・ログアウトのセッション切り替えを行います。
- **📱 PWA & SPA**: ネイティブアプリのようなボトムナビゲーションを実装。画面の表示を崩さずにタブ間（「マイページ」「マップ」「ガイド」）を高速に切り替えるUXを提供しています。

## 📂 主要なディレクトリ・ファイル構成

プロジェクトのフロントエンド機能は `src/` 配下に集約されており、以下の役割を持っています。

```text
src/
├── App.jsx                 # アプリのメインのルーティング定義（React Router）
├── firebase.js             # Firebase製品（Auth, Firestore, Storage）の初期化とエクスポート
├── index.css               # グローバルなデザイン定義、CSS変数、ベースコンポーネントのスタイル
├── context/
│   └── AuthContext.jsx     # Googleログインのセッション状態をアプリ全体に提供するContext API
├── components/
│   ├── Layout.jsx          # 全画面共通のヘッダーとボトムナビゲーションのレイアウト構造
│   └── Layout.css          # Layout専用のスタイリング
├── pages/
│   ├── Map.jsx             # [🗺️マップ] 地図描画、ピンのリアルタイム同期、最新リスト、フィルター表示の全機能
│   ├── Guide.jsx           # [📖ガイド] アプリのコンセプト、使い方、防災情報の折りたたみ表示
│   └── Profile.jsx         # [🐾マイページ] Googleログイン認証とプロフィール情報管理
├── utils/
│   └── imageUtils.js       # HEIC形式写真のJPEG変換や、画像圧縮処理のロジック
└── data/
    ├── shelters.json       # 避難所の初期マスターデータ
    └── guides.json         # アコーディオン内に表示する防災テキストデータ
```
